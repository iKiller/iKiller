Vim markdown-plugin setup with .vba document (windows 7)

  • 四 28 八月 2014
  • ·
  • misc

  1. Download markdown-plugin from: markdown-1.2.2.vba.gz

  2. Extract .gz document with 7zip(Or others software)

  3. Open markdown-1.2.2.vba with vim (Or Gvim)

  4. Run the cmd:

    :so %
:q

Now, enjoy vim editor with Syntax highlighting of markdown.
