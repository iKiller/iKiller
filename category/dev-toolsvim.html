ctags and python module(vim) work for autocomplete&tag-jump

设置 1. 设置python vim module，在vimrc文件中添加如下几行 ``` python << EOF import os import sys import vim for p in sys.path: if os.path.isdir(p): vim.command(r"set path+=%" % (p.replace(" ", r"\ "))) EOF ``` 准备tags文件 1. 运行ctags，生成tags文件，举例： >$ctags -R -f ...
